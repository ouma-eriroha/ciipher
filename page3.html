Ciipher fandom name:

As we mentioned before,Ciipher's meaning is a secret code and Ciipher members along with their fans will crack this code.
Before choosing the final fandom name,there was a pool where the fans will vote for the fandom name they like the most.
The options were :

-Liikey (Which is a combination between Like+Key and as Ciipher's name containes double i so the idea was to make the fandom name with double i )

-Clue (Which have the same meaning which is key , means the fans will be the clue that leads Ciipher).

The final result was "CLUE". So on June,22 2021 , Ciipher's fandom name was officially announced : "Clue" .
